# 🚦 AI Adaptive Traffic Signal System

## 📌 Overview
This project presents an AI-based adaptive traffic signal system that uses computer vision to detect vehicles and analyze traffic density in real time. Based on this analysis, the system dynamically adjusts traffic signal timing to improve traffic flow and reduce congestion.

The goal of this project is to provide a practical and low-cost solution for smarter traffic management in urban areas.

---

## 🚧 Problem Statement
Traffic signals in many cities operate on fixed timers and do not adapt to real-time traffic conditions. This leads to:
- Increased waiting time  
- Fuel wastage  
- Traffic congestion  

There is a need for a smarter system that can adjust signals dynamically based on actual traffic.

---

## 💡 Proposed Solution
This system uses an AI model to detect vehicles from video input and calculate traffic density. Based on vehicle count and density, it decides the signal timing automatically.

The system helps in:
- Reducing congestion  
- Improving traffic flow  
- Making traffic management more efficient  

---

## ⚙️ Features
- Real-time vehicle detection using YOLO  
- Traffic density calculation  
- Adaptive signal timing  
- Region of Interest (ROI) based analysis  
- Visual display of signal status  
- Can be extended for emergency vehicle prioritization  

---

## 🔄 Working Process
1. Video input is taken from traffic footage  
2. AI model detects vehicles in each frame  
3. System counts vehicles and calculates density  
4. Decision engine determines signal timing  
5. Output is displayed with signal status
